Elements

  • <small> element (and the .small class)
    • Used to create a smaller, secondary text in any heading

  • <mark> element (and the .mark class)
    • It will style <mark> and .mark with a yellow background color and some padding

  • <.abbr> element (and the .abbr class)
    • It will style <abbr> element with a dotted border bottom and a cursor with question mark on hover

  • <blockquote> element (and the .blockquote class)
    • .blockquote class to a <blockquote> when quoting blocks of content from another source.
    • when naming a source, like "from WWF's website", use the .blockquote-footer class
